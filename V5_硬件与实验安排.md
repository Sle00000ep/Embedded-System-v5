## 一、V4 vs V5 核心差异

| 项目 | V4 (ZET6) | V5 (VET6) |
|------|-----------|-----------|
| 芯片型号 | STM32F103ZET6 | **STM32F103VET6** |
| 封装/引脚 | 144 脚 | **100 脚** |
| Flash/RAM | 512KB / 64KB | 512KB / 64KB |
| GPIO 端口 | PA-PG（7 组） | **PA-PE（5 组，无 PF/PG）** |
| 板载 LED | PF0-PF7（SW3 控制 VLED） | **PE0-PE15**（SW-0 控制 VLED） |
| 光敏电阻 | PA1（AD Switch） | PA1（SW-1 AD 控制） |
| WKUP 按键 | PA0 | PA0 |
| 调试串口 | PA9/PA10 (CH340G) | PA9/PA10 (CH340G) |
| SWD | PA13/PA14 | PA13/PA14 |
